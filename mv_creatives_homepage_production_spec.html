MV Creatives Visual Design Specification v1.0

Digital Foundation Architecture v1.0 • Platform Handoff Spec • June 2026
Approved Architecture Baseline
This specification defines layout components, responsive spacing, and micro-interactions for the complete V1 platform: Home, Services, Work, Studio, and Contact. Insights/Blog sections are strictly removed. The design system uses desaturated media and real project references only.

1. Design System Tokens & Theme Matrix

The design system utilizes **Direction C (Hybrid Theme)**, shifting between true-depth dark viewports for brand manifesto structures and warm, high-contrast light viewports for text-heavy capabilities, process maps, and numerical case studies.

1.3 Grid, Layout, & Spacing System

The site runs on a 12-column desktop layout (max-width: 1280px) with generous gutter padding (120px to 200px section heights) ensuring elements have spatial breathing room.

  • Desktop Breakpoint: 12 Columns, 1280px max-width, outer gutters 8-10% viewport width, inner gutters 24-32px.
  • Tablet Breakpoint: 4 Columns, 720px max-width, outer gutters 6vw, inner gutters 24px.
  • Mobile Breakpoint: 2 Columns, full-width screen structure, 24px side padding, inner gutters 16px.

1.4 Depth & Elevation Tokens

Depth is created through layered positioning and structural borders, not heavy drop shadows. No shadows are permitted in dark context sections.

  • Level 0 (Base): Flat. Used for text paragraphs, headers, and separator borders.
  • Level 1 (Lift): box-shadow: 0 2px 8px rgba(0, 0, 0, 0.04). Applied to default light cards.
  • Level 2 (Hover): box-shadow: 0 8px 24px rgba(0, 0, 0, 0.08). Applied to active elements.
  • Level 4 (Glow): rgba(92, 108, 250, 0.12) with 400px blur. Creates ambient glow backdrops.

2.3 Custom Cursor Tracker (Desktop)

Default cursor is hidden. A custom 8px indigo follower tracks coordinates smoothly. When hovering over buttons, nav links, or interactive elements, it expands into a 40px targeting ring. In the portfolio section, it expands to 56px with a text label inside.

3.17 Cross-Page Conversion Strategy & Funnel

Funnel Path: User lands on Home → navigates to Work portfolio → reads Case Study → checks Services deep-dives → verifies Studio leadership trust → completes Contact qualification form.

Performance Target: Time-to-Interactive (TTI) must sit below 3s on 4G. Cumulative Layout Shift (CLS) = 0.0.

4. Motion & Easing Curves

Animations operate under hardware-accelerated restrictions: only transform and opacity can animate. Layout properties (width, padding, margin) must remain static.

  • Standard Reveal: translateY(20px → 0), opacity 0 → 1, duration 600ms, easing cubic-bezier(0.16, 1, 0.3, 1).
  • Button hover scale: scale(1.0 → 1.02), duration 250ms, easing cubic-bezier(0.25, 1, 0.5, 1).
  • Odometer count: spring animation logic, duration 800ms.
